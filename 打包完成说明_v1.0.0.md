## 🎯 安装说明

### 安装程序版本
1. 双击运行对应的 Setup.exe 文件
2. 选择安装路径（可自定义）
3. 安装完成后会在桌面创建快捷方式
4. 首次启动会自动导入默认文言文库

### 便携版
1. 双击运行对应的 Portable.exe 文件
2. 无需安装，直接使用
3. 首次启动会自动导入默认文言文库
4. 所有数据保存在用户数据目录

## 💾 数据存储位置

用户数据存储在：
```
C:\Users\[用户名]\AppData\Roaming\文言文工具\
```

包含：
- `.initialized` - 初始化标记文件
- `init-data.json` - 初始化数据（包含默认库和重点字列表）
- 其他用户数据（通过 localStorage 存储）

## 🎯 用户体验流程

### 1. 下载安装程序
```
文言文工具-1.0.0-x64-Setup.exe (约 150 MB)
```

### 2. 双击运行
- Windows 可能显示 SmartScreen 警告
- 点击"更多信息" → "仍要运行"

### 3. 安装向导
1. **欢迎页面** - 介绍应用特性
2. **许可协议** - 阅读并同意
3. **选择路径** - 默认或自定义
4. **选择组件** - 全部安装
5. **开始菜单** - 创建快捷方式
6. **安装进度** - 显示进度条
7. **完成页面** - 可选立即启动

### 4. 首次启动
```
🚀 首次运行，开始初始化...
📦 发现预设配置文件，正在加载...
✓ 配置已加载
✓ 库数据已加载:
  - 3 个库
  - 1234 个义项
  - 5678 个例句关联
🎉 预设配置加载完成！
```

### 5. 开始使用
- 无需任何配置
- 所有功能立即可用
- API Key 已预设
- 库数据已加载

## 📊 技术规格

### 系统要求
- **操作系统**: Windows 7/8/10/11
- **架构**: x64 或 ia32
- **内存**: 至少 2GB RAM
- **磁盘**: 至少 200MB 可用空间

### 安装包信息
- **格式**: NSIS 安装程序
- **压缩**: 最大压缩
- **签名**: 可选代码签名
- **大小**: 约 150MB（含库数据）

### 性能优化
- **启动速度**: < 3秒
- **安装速度**: < 2分钟
- **内存占用**: < 200MB
- **CPU占用**: < 5%（空闲时）

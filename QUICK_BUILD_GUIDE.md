# 🚀 快速打包指南

## 一键打包（推荐）

```bash
npm run build:with-config
```

就这么简单！脚本会自动：
- ✅ 检查你的配置文件
- ✅ 生成漂亮的安装界面
- ✅ 打包成 Windows 安装程序

## 前提条件

确保你已经导出了配置文件：

1. 打开应用 → 设置页面
2. 点击 **"💾 导出全部"**
3. 文件会保存到 `dist/config-custom-[时间戳].json`

## 输出结果

打包完成后，在 `dist/` 目录下会有：

```
文言文工具-1.0.0-x64-Setup.exe      # 64位安装程序
文言文工具-1.0.0-ia32-Setup.exe     # 32位安装程序
```

## 常见问题

**Q: 配置文件在哪里？**
A: `dist/config-custom-[时间戳].json`

**Q: 如何更新配置？**
A: 重新导出配置，然后重新打包

**Q: 安装包太大？**
A: 这是正常的，因为包含了所有库数据

**Q: 如何自定义图标？**
A: 替换 `build/icon.ico` 文件

# 配置目录说明

此目录用于存放应用的长期配置文件。

## 目录结构

```
config/
├── app-config.json    # 应用配置（AI设置、界面设置等）
├── libraries.json     # 库数据（文章、义项、关联等）
└── README.md          # 本说明文件
```

## 配置文件说明

### app-config.json

应用的主配置文件，包含：
- AI 配置（API Key、模型、并发设置等）
- 界面设置（主题、背景等）
- 功能开关
- 教程播放记录

### libraries.json

库数据文件，包含：
- 文章库（libraries）
- 义项（definitions）
- 字-义项关联（characterDefinitionLinks）
- 短句（shortSentences）
- 重点字（keyCharacters）

## 使用说明

1. **自动同步**：程序启动时会将此目录的配置复制到 `temp` 目录
2. **热更新**：运行时修改此目录的文件，程序会自动检测并更新
3. **程序内修改**：在程序设置页面的修改会自动同步到此目录
4. **备份**：建议定期备份此目录的文件

## 注意事项

- 请勿在程序运行时直接编辑 `temp` 目录的文件
- 修改配置文件时请确保 JSON 格式正确
- 如果配置文件损坏，可以删除后重启程序，会自动生成默认配置

# MiniMax-M2 快速开始指南

## 现在系统使用什么模型？

✅ **MiniMaxAI/MiniMax-M2** - 通过硅基流动API调用

## 我需要做什么？

**什么都不需要做！** 系统已经配置好了，直接使用即可。

## 验证是否在使用MiniMax-M2

### 方法1：查看控制台日志
1. 打开浏览器开发者工具（F12）
2. 进入"控制台"标签
3. 执行任何AI操作（生成义项或短句）
4. 查看日志输出：
   ```
   [AI请求] 使用: MiniMaxAI/MiniMax-M2
   [短句生成] 使用 MiniMax-M2 模型
   ```

### 方法2：查看源代码
- `src/services/ai.ts` - 第16行：`model: 'MiniMaxAI/MiniMax-M2'`
- `src/services/shortSentence.ts` - 第7行：`const MODEL = 'MiniMaxAI/MiniMax-M2'`

## 性能提升

## 如何调整并发参数？

1. 进入管理页面
2. 点击"⚙️ 设置"标签
3. 点击"打开并发参数设置"
4. 根据推荐值调整：
   - AI义项并发：6（提高到6）
   - 短句生成并发：14
   - 批次间延迟：150ms
   - 重试延迟：400ms

## 常见问题

### Q: 为什么日志显示MiniMax-M2但我之前用的是GLM？

1. 检查网络连接
2. 查看浏览器控制台的错误信息

### Q: 可以切换回GLM吗？
A: 可以，但不推荐。MiniMax-M2性能更好。
